### VideoHub Platform

A full-stack video-sharing platform inspired by modern content platforms and video streaming services.

The project was built to explore real-world backend challenges such as media streaming, transactional consistency, authentication, content management, and scalable interaction tracking.

#### Highlights

* Native HTML5 video streaming using HTTP 206 Partial Content
* Support for both local video uploads and YouTube embed integration
* JWT authentication with extensible OAuth architecture
* Atomic likes and subscriptions implemented through database transactions
* Optimized view counting using buffered writes to reduce database load
* Video upload, editing, search, and content management
* Threaded comments system
* User profiles with avatars, banners, and channel customization
* Separation of business logic, data access, and API layers

#### Stack

Node.js • Express.js • TypeScript • Prisma ORM • PostgreSQL • JWT • Multer • Next.js • React • Axios • TailwindCSS • React Player
